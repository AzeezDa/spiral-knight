# Spiral Knight

Inspired by the [Numberphile video](https://youtu.be/UiX4CFIiegM) and the [extra video](https://youtu.be/VgmDuBCayPw) about placing knights around a spiral on a checkboard such that no knight is threatened by a previously placed knight.
This is a website that allows the creations of such boards using different-sized knights (Wazir, Antelope, etc.) as well as different colours.
The images may be saved by right-clicking and saving!

More can be found in the deployment [here](https://azeezda.github.io/spiral-knight/).
